# 🚀 GODSAENG - AI 기반 라이프 트래킹

AI 기반 일상 기록 및 분석 웹 애플리케이션

## 🎯 주요 기능

- ✅ 사용자 인증 (회원가입/로그인)
- ✅ 일상 기록 작성 (활동, 감정, 태그)
- ✅ AI 기반 일일 분석 리포트
- ✅ 기록 조회 및 관리
- ✅ 무료 AI API 지원 (Hugging Face, Gemini, Ollama)

## 🚀 빠른 시작

### 로컬 개발

```bash
# 백엔드
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

# 프론트엔드
cd frontend
npm install
npm run dev
```

### 클라우드 배포 (무료)

**5분 안에 배포하기:**
```bash
cat DEPLOY_NOW.md
```

**배포 가이드:**
- `DEPLOY_NOW.md` - 빠른 배포 가이드
- `FINAL_DEPLOYMENT_GUIDE.md` - 상세 배포 가이드
- `QUICK_DEPLOY.md` - 5분 빠른 배포
