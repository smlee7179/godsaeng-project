# 변경 이력

## [1.0.0] - 2024-11-01

### 추가
- FastAPI 백엔드 구현
- React 프론트엔드 구현
- 사용자 인증 시스템 (JWT)
- 라이프 기록 CRUD 기능
- AI 분석 기능 (OpenAI GPT API)
- MongoDB 데이터베이스 연동
- Docker 및 Docker Compose 설정
- 배포 스크립트 및 가이드

### 기능
- 회원가입/로그인
- 일상 기록 작성/수정/삭제
- 기록 목록 및 상세 조회
- AI 일상 분석 리포트
- 대시보드 통계

### 기술 스택
- Backend: FastAPI, Python, MongoDB
- Frontend: React, Vite, React Router
- Deployment: Docker, Docker Compose
